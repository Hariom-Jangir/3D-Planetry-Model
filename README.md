🪐 3D Planetary Model
Design Credit Course Project
A web-based 3D interactive planetary model built using Three.js. This project is currently Under Development as part of a Design Credit course.
🚀 Overview
This project aims to create a visually engaging and scientifically inspired 3D model of [a specific planet / the solar system]. Users can interact with the 3D environment to explore planetary textures, orbits, and atmospheric effects directly in their browser.
✨ Key Features (Planned & Current)
Interactive Controls: Smooth rotation and zooming using OrbitControls.
Realistic Textures: High-resolution mapping for planetary surfaces.
Dynamic Lighting: Implementation of PointLight to simulate a sun/star light source.
Atmospheric Glow: (Optional) Custom shaders for planetary atmospheres.
Responsive Design: Optimized for different screen sizes using [Vite or WebGLRenderer updates].
🛠️ Tech Stack
Engine: Three.js (JavaScript 3D Library).
Bundler: Vite (Recommended for fast development).
Language: JavaScript / [TypeScript].
Assets: NASA-inspired planetary textures.
