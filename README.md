# 🪐 3D Planetary Model  
**Design Credit Course Project**

A web-based 3D interactive planetary model built using Three.js. This project is currently **Under Development** as part of a Design Credit course.

---

## 🌐 Live Demo  
👉 https://hariom3dplanetrymodel.vercel.app/

---

## 🚀 Overview  
This project aims to create a visually engaging and scientifically inspired 3D model of the solar system. Users can interact with the 3D environment to explore planetary textures, orbits, and atmospheric effects directly in their browser.

---

## ✨ Key Features (Planned & Current)

- **Interactive Controls**: Smooth rotation and zooming using OrbitControls  
- **Realistic Textures**: High-resolution mapping for planetary surfaces  
- **Dynamic Lighting**: Implementation of PointLight to simulate a sun/star light source  
- **Responsive Design**: Optimized for different screen sizes using Vite/WebGLRenderer  

---

## 🛠️ Tech Stack  

- **Engine**: Three.js (JavaScript 3D Library)  
- **Bundler**: Vite  
- **Language**: JavaScript  
- **Assets**: NASA-inspired planetary textures  

---

## 🏗️ Getting Started  

### 📌 Prerequisites  
Make sure you have **Node.js** installed.

### ⚙️ Installation  

Clone the repository:
```bash
git clone https://github.com/Hariom-Jangir/3D-Planetry-Model.git
cd 3D-Planetry-Model
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open in browser:
```
http://localhost:5173
```
